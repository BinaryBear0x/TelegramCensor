# TelegramCensor

TelegramCensor, belirli kelimeleri içeren mesajları tespit edip silen ve sansürlenmiş sürümünü yeniden paylaşan bir Telegram bot örneğidir.

## Ne Yapar

- yasaklı kelime listesini dosyadan okur
- gelen mesajları kontrol eder
- eşleşme bulunursa mesajı siler
- sansürlenmiş içeriği gruba geri yollar

## Teknolojiler

- Python
- python-telegram-bot
- düzenli ifadeler

## Notlar

- Bot token bilgisinin `bot.py` içinde doldurulması gerekir
- Kelime listesi `küfür.txt` dosyasından okunur
